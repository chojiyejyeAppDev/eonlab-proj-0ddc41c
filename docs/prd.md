# 추론 탐정 (Inference Detective)

## 문제 정의 (Problem Statement)

수능 비문학 영역에서 요구되는 추론적 독해력은 단순 암기나 반복 풀이로는 길러지지 않는다. 기존 교육 시장은 정답 중심의 문제풀이 콘텐츠로 포화 상태이며, **추론 과정 자체를 훈련**하도록 설계된 디지털 콘텐츠는 전무하다. 비문학을 처음 접하는 중고등학생들은 "왜 이 답이 맞는가"에 대한 논리적 사고 과정을 경험하지 못한 채 결과 중심 학습에 머문다. 본 제품은 추론 경로를 직접 조립하고 검증하는 게임 방식을 통해 논리적 사고력의 기초를 형성하고, **B2C 프리미엄 모델**을 통해 시장 수요를 빠르게 검증한다.

| 구분 | 내용 |
|------|------|
| **핵심 타겟** | 비문학을 처음 접하는 중학생 ~ 고등학교 1-2학년 |
| 학습 수준 | 추론 개념을 체계적으로 학습한 경험이 없는 입문자 |

| **유료 구독자** | 일일 제한 없이 전 레벨 콘텐츠를 활용하는 적극적 학습자 |

## 수익 모델 (Monetization Model)

### 구조: 콘텐츠 제한형 프리미엄 (하루 N개 방식)

```
[무료 사용자]                    [유료 구독자]
─────────────                   ─────────────
하루 5개 문제 제공               무제한 문제 풀기
Level 1-7 모두 접근 가능         전체 힌트 시스템 이용
기본 힌트만 제공                 성장 대시보드 전체 열람
오답 분석 미제공                 오답 패턴 분석 제공
                                월간 신규 콘텐츠 우선 접근
```

### 일일 한도 소진 시 경험 설계: **보상형 해제 (B안)**

무료 사용자가 하루 5개 문제를 모두 소진했을 때:

> "오늘 분량을 다 사용했어요! 아래 방법으로 문제를 더 풀 수 있어요."
>
> - 📣 **친구 초대** → 추가 문제 2개 즉시 해제 (초대받은 친구도 +2개 보너스)
> - 📢 **광고 시청** (15-30초) → 추가 문제 1개 해제 (하루 최대 2회)
> - ⭐ **구독 전환** → 지금 바로 무제한으로 계속하기

**설계 의도:** 수능을 준비하는 학생은 "차단"에 민감하게 반응할 수 있으므로, 강제 차단보다 **선택지를 주는 방식**으로 거부감을 낮추면서 바이럴과 유료 전환을 동시에 유도한다.

### 구독 요금 (예시)
| 플랜 | 가격 | 혜택 |
|------|------|------|
| 월간 구독 | 9,900원/월 | 무제한 + 전체 기능 |
| 연간 구독 | 79,200원/년 | 월간 대비 33% 할인 |
| 학생 인증 할인 | 6,900원/월 | 재학생 인증 시 적용 |

## 핵심 기능 (Core Features)

### 1. 핵심 게임플레이: 추론 경로 조립 (탐정 모드)

- 수능/모의고사 난이도의 비문학 지문 제공
- 지문 하단에 **결론 문장** 제시 (텍스트에 명시되지 않은 논리적 파생 명제)
- 플레이어가 지문 내 문장들을 드래그&드롭으로 경로 슬롯에 배치
- **3-7단계 추론 체인**을 완성하는 것이 목표
- 배치 중 문장 간 논리 연결 강도를 실시간 시각화

```
[지문 영역]                        [추론 경로 조립 영역]
─────────────────                  ──────────────────────────────
문장A  문장B  문장C    →   슬롯1 → 슬롯2 → 슬롯3 → [결론]
문장D  문장E  문장F        🟢강함  🟡보통  🔴약함
문장G  문장H  ...          ↑ 드래그&드롭으로 배치
```

---

### 2. 7단계 레벨 시스템

| 레벨 | 단계 | 추론 단계 수 | 특징 |
|------|------|-------------|------|
| 1 | 입문 | 3단계 | 단순 인과관계, 일상 소재 |
| 2 | 기초 | 3-4단계 | 부분적 추상 개념 포함 |
| 3 | 기초-중급 | 4단계 | 수식어/종속절 포함 문장 |
| 4 | 중급 | 4-5단계 | 실전 모의고사 수준 |
| 5 | 중상급 | 5-6단계 | 복잡한 논리 구조 (대조/역전) |
| 6 | 고급 | 6단계 | 수능 실전 난이도 |
| 7 | 마스터 | 6-7단계 | 고난도 추상 지문, 힌트 없음 |

- **레벨업 조건:** 연속 3세션 정확도 80% 이상
- 레벨업 시 **잠금 해제 애니메이션** + 성취 피드백 제공
- 무료 사용자도 모든 레벨에 접근 가능 (단, 하루 5개 문제 제한 적용)

---

### 3. 단계별 힌트 시스템 (핵심 학습 장치)

레벨이 올라갈수록 힌트가 **점점 간접적**으로 변화 (추론 능력 자립이 목표):

| 레벨 | 힌트 유형 | 내용 |
|------|-----------|------|
| 1-2 | **직접 힌트** | "여기엔 원인을 나타내는 문장이 필요해요" + 문장 하이라이트 |
| 3-4 | 반직접 | "앞 문장의 결과를 설명하는 문장이 필요해요" (하이라이트 없음) |
| 5-6 | 간접 | "논리 접속어(따라서, 그러므로)를 찾아보세요" |
| 7 | 힌트 없음 | 완전 자력으로 완성 |

- 힌트는 **힌트 포인트** 소모 (플레이를 통해 획득)
- **유료 구독자:** 힌트 포인트 2배 획득 + 힌트 사용 기록 상세 분석 제공
- **무료 사용자:** 기본 힌트 포인트만 제공, 상세 분석 미제공

---

### 4. 실시간 논리 연결 피드백

- 슬롯에 문장 배치 시 논리 연결 강도를 실시간 시각화:
  - 🟢 강한 연결 / 🟡 약한 연결 / 🔴 논리 단절
- 경로 제출 후: **단계별 오류 상세 설명** 제공
- 오류 설명은 정답 제시보다 **"왜 이 문장이 여기에 있어야 하는가"** 에 집중

---

### 5. 추론 성장 대시보드 (유료 전용)

- 정확도 추이 그래프 (레벨별/기간별)
- 자주 틀리는 논리 오류 패턴 분석 (예: "역인과 오류 빈번", "대조 관계 식별 약함")
- 힌트 의존도 추이 시각화 (힌트 사용 감소 = 성장 가시화)
- 추론 스타일 리포트: 귀납/연역/유추 추론 강점 파악

> **무료 사용자**에게는 대시보드 일부(정확도 요약만)를 미리보기로 제공하여 유료 전환 동기 유발

### 6. 바이럴 루프 설계

```
[하루 5개 소진]
       ↓
[친구 초대 시 +2개 해제]
       ↓
[초대받은 친구: 가입 + 보너스 문제]
       ↓
[신규 사용자의 일일 루프 시작]
       ↓
[반복 → 유료 전환 압력 자연 발생]
```

- 초대 링크에 초대자 닉네임/레벨 포함 → 소속감 및 경쟁심 자극
- "친구도 탐정이 됐어요! 지금 레벨 2예요" 알림으로 재방문 유도
