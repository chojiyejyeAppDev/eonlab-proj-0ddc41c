## ✅ Technical SEO Checklist

- [ ] Core Web Vitals 최적화 (LCP < 2.5s)
- [ ] 모바일 반응형 디자인 적용
- [ ] Schema Markup (EducationalApplication) 삽입
- [ ] Sitemap.xml 및 robots.txt 설정
- [ ] Open Graph / Twitter Card 메타태그
- [ ] 페이지 로딩 속도 최적화 (이미지 WebP 변환)
- [ ] HTTPS 적용 및 보안 인증서 확인
- [ ] 내부 링크 구조 최적화
